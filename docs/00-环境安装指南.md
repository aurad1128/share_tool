# 开发环境安装指南（Python版）

## 必需软件清单

开发"面对面传输工具"需要安装以下软件：

1. ✅ **Python** - Python编程语言（必需）
2. ✅ **Git** - 版本控制工具（推荐）
3. ✅ **VS Code** 或 **PyCharm** - 代码编辑器（推荐）

---

## 1. 安装 Python

### 为什么需要Python？
Python是一种简单易学的编程语言，我们的项目需要用它来：
- 编写爬虫程序
- 开发桌面应用界面
- 处理数据和生成Excel
- 打包成Windows应用程序

### 下载和安装步骤

#### 方法一：官网下载（推荐）

1. **访问Python官网**
   - 打开浏览器，访问：https://www.python.org/downloads/
   
2. **下载安装包**
   - 点击"Download Python 3.11.x"按钮（推荐3.10或3.11版本）
   - 文件名类似：`python-3.11.x-amd64.exe`

3. **运行安装程序**
   - 双击下载的 `.exe` 文件
   - **⚠️ 重要：勾选"Add Python to PATH"（必须勾选！）**
   - 选择"Install Now"（推荐）或"Customize installation"
   - 等待安装完成

4. **验证安装**
   - 按 `Win + R` 键
   - 输入 `cmd` 并回车
   - 在命令窗口输入：
   ```bash
   python --version
   ```
   - 如果显示版本号（如 `Python 3.11.5`），说明安装成功

5. **验证pip**
   - 在同一个命令窗口输入：
   ```bash
   pip --version
   ```
   - 如果显示版本号（如 `pip 23.2.1`），说明pip也安装成功

### 安装后配置（推荐）

#### 配置pip国内镜像（加速下载）
```bash
# 使用清华大学镜像
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# 或使用阿里云镜像
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
```

#### 验证镜像配置
```bash
pip config list
```

#### 升级pip到最新版本
```bash
python -m pip install --upgrade pip
```

## 2. 安装 Git（推荐）

### 为什么需要Git？
Git用于代码版本管理，可以：
- 保存代码历史记录
- 回退到之前的版本
- 备份代码

### 下载和安装步骤

1. **访问Git官网**
   - 打开浏览器，访问：https://git-scm.com/download/win

2. **下载安装包**
   - 点击"64-bit Git for Windows Setup"
   - 文件名类似：`Git-2.x.x-64-bit.exe`

3. **运行安装程序**
   - 双击下载的 `.exe` 文件
   - 一路点击"Next"（使用默认设置即可）
   - 点击"Install"
   - 等待安装完成

4. **验证安装**
   - 打开命令窗口（cmd）
   - 输入：
   ```bash
   git --version
   ```
   - 如果显示版本号，说明安装成功

5. **配置Git用户信息**
   ```bash
   git config --global user.name "你的名字"
   git config --global user.email "你的邮箱"
   ```

## 3. 安装 VS Code 或 PyCharm（推荐）

### 选项A：VS Code（轻量级，推荐初学者）

#### 为什么选择VS Code？
- 免费开源
- 轻量快速
- 插件丰富
- 适合Python开发

#### 下载和安装步骤

1. **访问VS Code官网**
   - 打开浏览器，访问：https://code.visualstudio.com

2. **下载安装包**
   - 点击"Download for Windows"
   - 文件名类似：`VSCodeUserSetup-x64-1.x.x.exe`

3. **运行安装程序**
   - 双击下载的 `.exe` 文件
   - 接受协议
   - **重要：勾选以下选项：**
     - ✅ 将"通过Code打开"操作添加到Windows资源管理器文件上下文菜单
     - ✅ 将"通过Code打开"操作添加到Windows资源管理器目录上下文菜单
     - ✅ 将Code注册为受支持的文件类型的编辑器
     - ✅ 添加到PATH
   - 点击"安装"
   - 等待安装完成

4. **安装Python插件**
   打开VS Code后，按 `Ctrl + Shift + X` 打开插件市场，搜索并安装：
   - **Python** - 微软官方Python插件（必需）
   - **Pylance** - Python语言服务器（必需）
   - **Chinese (Simplified)** - 中文语言包（可选）
   - **Python Indent** - 自动缩进（推荐）
   - **autoDocstring** - 自动生成文档（推荐）

### 选项B：PyCharm（功能强大，适合专业开发）

#### 为什么选择PyCharm？
- 专为Python设计
- 智能代码提示
- 强大的调试功能
- 内置虚拟环境管理

#### 下载和安装步骤

1. **访问PyCharm官网**
   - 打开浏览器，访问：https://www.jetbrains.com/pycharm/download/

2. **选择版本**
   - **Community Edition（社区版）** - 免费，功能够用
   - Professional Edition（专业版）- 付费，功能更多

3. **下载并安装**
   - 下载Community Edition
   - 双击安装文件
   - 一路点击"Next"
   - 勾选"Add launchers dir to the PATH"
   - 完成安装

## 常见问题

### Q1: 命令提示"python不是内部或外部命令"怎么办？
**A:** 安装Python时没有勾选"Add Python to PATH"。
解决方法：
1. 卸载Python
2. 重新安装，记得勾选"Add Python to PATH"
3. 或者手动添加环境变量

### Q2: pip下载速度很慢怎么办？
**A:** 配置国内镜像：
```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3: 安装过程中遇到权限问题？
**A:** 右键点击安装程序，选择"以管理员身份运行"。

### Q4: Python版本太低怎么办？
**A:** 卸载旧版本，重新下载最新的3.10或3.11版本安装。

### Q5: 如何卸载Python？
**A:** 
1. 打开"控制面板" → "程序和功能"
2. 找到"Python 3.x.x"
3. 右键点击"卸载"

### Q6: pip install时出现"error: Microsoft Visual C++ 14.0 is required"？
**A:** 某些包需要C++编译器。解决方法：
1. 下载并安装"Microsoft C++ Build Tools"
2. 或者使用预编译的wheel文件

### Q7: VS Code无法识别Python？
**A:** 
1. 按 `Ctrl + Shift + P`
2. 输入"Python: Select Interpreter"
3. 选择已安装的Python版本
